# Repositorio Pariwana

Aplicacion interna de Pariwana Hostels para gestionar horarios de trabajadores y generar el reporte XLSX de carga de turnos para BUK.

## Estado

Repositorio inicial. La estructura de backend, base de datos y exportacion BUK debe implementarse por fases pequenas y verificables.

## Notas de seguridad

- No subir archivos `.env`.
- No hardcodear credenciales de Supabase, GitHub u otros servicios.
- Las llaves secretas deben vivir solo en variables de entorno del backend o en secrets del proveedor de despliegue.
